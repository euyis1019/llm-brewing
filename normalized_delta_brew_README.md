## 定义

- **ΔBrew** = FJC_layer − FPCL_layer（绝对层数）
- **Normalized ΔBrew** = ΔBrew / n_layers
- 仅在 FJC≠null 的 sample 上计算（coverage = Resolved% + Overprocessed%）

## Key Observations

- 所有 16 个模型 ΔBrew > 0 — 信息总是先 probe 可读，后 CSD 可用
- Normalized ΔBrew 范围 0.24–0.42，即模型用约 24%–42% 的层完成"可读→可用"转换
- 同模型内 value_tracking 的 ΔBrew 通常最大，computing/loop 最小
- ΔBrew 存在选择偏差：高 unresolved 率的任务/模型中，仅较简单的子集参与计算
