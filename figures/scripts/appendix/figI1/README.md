# Fig I.1 — Difficulty Scaling: Resolved vs Unresolved

**Caption**: Difficulty scaling: Resolved % (green) and Unresolved % (gray) along each task's primary difficulty dimension. Each panel shows one task with 3 difficulty levels.

- Value Tracking: depth (1→3)
- Computing: steps (2→4)
- Conditional: depth (1→3)
- Function Call: depth (1→3)
- Loop: iterations (2→4)
- Loop (Unrolled): iterations (2→4)
