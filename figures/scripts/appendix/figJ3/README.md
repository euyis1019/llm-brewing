# Fig J.3 — FPCL vs Resolved %

**Caption**: FPCL (normalized) vs Resolved % across 16 models × 6 tasks (96 points). Density clouds show per-family clustering. Regression line with 95% CI band. The weak correlation illustrates the "information paradox": early information availability does not guarantee successful resolution.
